01

Exercise 1 — Even or Odd Printer

Ask the user for a number n, then print numbers from 1 to n and state whether each is even or odd.

solution1.py 
n = int(input("Enter a number: "))
for i in range(1, n + 1):
    if i % 2 == 0:
        print(i, "Even")
    else:
        print(i, "Odd")
1 Take input from user and convert to integer
2 Loop from 1 to n — use n + 1 because range() excludes the stop value
3 Check if remainder after dividing by 2 is 0 — if yes, number is even
02

Exercise 2 — Sum of Numbers

Calculate the sum of numbers from 1 to n.

solution2.py 
n = int(input("Enter a number: "))
total = 0
for i in range(1, n + 1):
    total = total + i
print("Total:", total)

We use an accumulator variable (total = 0) that starts at 0 and grows with each loop iteration. If n = 3: total = 0+1 = 1, then 1+2 = 3, then 3+3 = 6.

04

Exercise 4 — Count Positive and Negative Numbers

Take 5 numbers and count how many are positive and how many are negative.

solution4.py 
positive = 0
negative = 0
for i in range(5):
    num = int(input("Enter a number: "))
    if num >= 0:
        positive += 1
    else:
        negative += 1
print("Positive:", positive)
print("Negative:", negative)

positive += 1 is shorthand for positive = positive + 1. We use two counter variables that start at 0 and increment as needed.

05

Exercise 5 — Skip Multiples of 3

Print numbers 1 to 20, skipping multiples of 3.

solution5.py 
for i in range(1, 21):
    if i % 3 == 0:
        continue
    print(i)

continue skips the rest of the current loop iteration when the number is divisible by 3.

06

Exercise 6 — Number Guessing Game

Keep asking the user until they guess the correct secret number.

solution6.py 
secret = 7
while True:
    guess = int(input("Enter your guess: "))
    if guess == secret:
        print("Correct!")
        break
    else:
        print("Try again")

while True creates an infinite loop that only stops when the user guesses correctly and break is reached.

07

Exercise 7 — Password with Limited Attempts

Allow only 3 password attempts.

solution7.py 
password = "admin123"
for i in range(3):
    user_input = input("Enter password: ")
    if user_input == password:
        print("Access Granted")
        break
else:
    print("Access Denied")
for-else pattern Python's for loop supports an else block. The else runs only if the loop completed without hitting a break. Here, it runs if all 3 attempts were used up without a correct password.

Mini Project 1 — Student Marks Analyzer

Ask user to enter 5 marks, store them in a list, and calculate total, average, pass count, and fail count.

marks_analyzer.py 
marks = []
for i in range(5):
    mark = int(input("Enter mark: "))
    marks.append(mark)

total = 0
pass_count = 0
fail_count = 0
for mark in marks:
    total += mark
    if mark >= 50:
        pass_count += 1
    else:
        fail_count += 1

average = total / len(marks)
print("Total:", total)
print("Average:", average)
print("Pass:", pass_count)
print("Fail:", fail_count)

len(marks) gives the total number of items in the list. marks.append(mark) adds each mark to the list.

09

Mini Project 2 — Simple ATM Menu

atm.py 
balance = 1000
while True:
    print("1. Check Balance")
    print("2. Deposit")
    print("3. Withdraw")
    print("4. Exit")
    choice = input("Enter choice: ")
    if choice == "1":
        print("Balance:", balance)
    elif choice == "2":
        amount = int(input("Enter deposit amount: "))
        balance += amount
    elif choice == "3":
        amount = int(input("Enter withdrawal amount: "))
        if amount <= balance:
            balance -= amount
        else:
            print("Insufficient balance")
    elif choice == "4":
        print("Goodbye")
        break
    else:
        print("Invalid choice")

The menu repeats using while True. Each option is handled using if/elif/else. The program exits only when the user chooses 4 and break is executed.

Lesson Summary

  • Accumulator pattern: start a variable at 0 and add to it in each loop iteration
  • Counter pattern: start at 0 and increment by 1 when a condition is met
  • list.append() adds items to a list; len(list) gives the count of items
  • while True with break is useful for menus and games where you don't know how many iterations are needed
  • The for...else pattern runs the else block only if the loop finished without break
