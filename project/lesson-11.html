Lesson 11

Decision Making in Python

Until now, your programs could only run step by step. From this lesson onward, your programs can make decisions using if, else, and elif.

By the end of this lesson you will be able to

  • Write decision-making programs using if, elif, and else
  • Use comparison operators such as >, <, ==, !=, >=, <=
  • Combine conditions using logical operators and and or
  • Solve conditional problems with programmatic logic
01

Why Do We Need Decision Making?

In real life, we make decisions all the time. Since computer programs simulate and automate real-life scenarios, they also need this decision-making ability.

  • if it rains → take an umbrella
  • if you are hungry → eat food
  • if marks are above 50 → pass
02

The if Statement

The simplest form of decision making. The indented block only runs when the condition is True.

syntax 
if condition:
    # code runs if condition is True
example.py 
age = 20
if age > 18:
    print("You are an adult")
Output
You are an adult

How it works:

1 Python evaluates the condition: age > 18
2 If it is True, the indented code runs
3 If it is False, it is skipped entirely
03

Comparison Operators

Conditions are usually comparisons. Python has six comparison operators:

Operator Meaning Example
==Equal toa == b
!=Not equal toa != b
>Greater thana > b
<Less thana < b
>=Greater than or equal toa >= b
<=Less than or equal toa <= b
04

Indentation and Code Blocks

After the if keyword and the colon :, every line that belongs to that block must be indented. Python uses indentation to define where a block begins and ends — not curly braces like Java or C.

Correct 
if True:
    print("This works")
Wrong — IndentationError 
if True:
print("Error!")
Python vs other languages Languages like Java use curly braces { }, so indentation there is optional style. In Python, indentation is mandatory — it is the only way Python knows which code belongs to which block.
05

Using else

Use else to run a different block of code when the condition is false.

example.py 
age = 16

if age > 18:
    print("Adult")
else:
    print("Not an adult")
Output
Not an adult
Think about it — what if age is exactly 18? The condition age > 18 is strictly greater than. So age 18 prints "Not an adult". If you want to include 18-year-olds as adults, use age >= 18 instead.
06

Using elif — Multiple Conditions

When you have more than two possible outcomes, use elif (short for else if) to chain conditions.

grades.py 
marks = int(input("Enter your marks: "))

if marks >= 75:
    print("Distinction")
elif marks >= 50:
    print("Pass")
else:
    print("Fail")
1The first condition is checked. If the user enters 80, then 80 >= 75 is True — it prints Distinction and stops.
2If the user enters 60, the first condition is False. The second condition 60 >= 50 is True — it prints Pass.
3If the user enters 30, both conditions are False. The else block runs — it prints Fail.
07

Comparing Strings

You can compare text (strings) using == the same way you compare numbers.

login.py 
password = input("Enter the password: ")

if password == "John123":
    print("Logged in successfully")
else:
    print("Password incorrect")
Python is case-sensitive "John123" and "john123" are completely different strings. Entering john123 (lowercase j) will always print "Password incorrect".
08

Combining Conditions — and & or

and — both conditions must be True for the block to run.

age = 20
has_id = True

if age >= 18 and has_id:
    print("Entry allowed")

# Both requirements must be met

or — at least one condition must be True.

if age >= 18 or has_id:
    print("Entry allowed")

# Either requirement is enough
09

Complete Example — Grade Calculator

Let's build a program where the user enters marks and the program decides the grade. It should also print "Invalid marks" if the user enters a value above 100, since that is the maximum possible score.

GradeMark range
A75 – 100
B65 – 74
C50 – 64
S35 – 49
F0 – 34
grade_calc.py 
marks = int(input("Enter your marks: "))

if marks > 100:
    print("Invalid marks")
elif marks >= 75:
    print("A")
elif marks >= 65:
    print("B")
elif marks >= 50:
    print("C")
elif marks >= 35:
    print("S")
else:
    print("Fail")
Try it yourself Run this program and enter different values — including numbers above 100. Observe which branch executes each time.

Common Beginner Mistakes

Using = instead of ==

A single = is assignment. A double == is comparison.

Wrong 
if x = 5:   # SyntaxError
Correct 
if x == 5:
Forgetting indentation
Wrong 
if x > 5:
print("Hello")  # IndentationError
Correct 
if x > 5:
    print("Hello")
Comparing input without converting to int

input() always returns a string. You cannot compare a string with a number — you must convert it first using int().

Wrong 
age = input("Enter age: ")
if age > 18:  # TypeError
Correct 
age = int(input("Enter age: "))
if age > 18:
