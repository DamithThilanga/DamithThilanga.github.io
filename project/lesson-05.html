Lesson 5

Variables, Data Types, and Algorithms

No matter how simple or complex a program is, it always depends on three fundamental ideas: variables, data types, and algorithms. If you understand these clearly, everything else in programming becomes much easier.

By the end of this lesson you will be able to

  • Explain what a variable is and how it stores data in a program
  • Identify and use the four basic Python data types: int, float, str, and bool
  • Convert between data types using type conversion functions
  • Define what an algorithm is and explain its role in programming
01

Variables

A variable is a name used to store a value in a program. Think of it like a labeled container — you can put values into it, remove values, and change the values. Instead of asking the program to remember values directly, you give them names (labels) so you can use those names throughout the program.

example.py 
marks = 75

Here, marks is the variable name and 75 is the value stored in it. Later in the program, you can use marks instead of writing 75 again and again. If you want to change the marks to 85, simply write marks = 85 — one change affects everywhere it is used.

Variables make programs:

  • Easier to read
  • Easier to update
  • More flexible
02

Data Types

A data type tells the computer what kind of value a variable is storing. This is important because different types of data are handled in different ways.

1. Integer (int) — Whole numbers without decimal points.

age = 30
marks = 80
total = 10 + 15

2. Float (float) — Numbers with decimal points, used when precision is needed.

price = 99.99
temperature = 36.5
bmi = 67.5 / (1.8 * 1.8)

3. String (str) — Text values, which must be inside quotes. Python allows both single and double quotes.

name = "John"
city = 'Colombo'
message = "Hello World"
grade = "A"
full_name = "John " + "Perera"  # concatenation

4. Boolean (bool) — Represents only two values: True or False (and also None, which indicates no value has been stored).

is_passed = True
availability = False
is_logged_in = True
03

Why Are Data Types Important?

Data types help the computer to store data correctly, perform correct operations, and avoid errors. For example:

  • You can add numbers: 5 + 3
  • But adding text and numbers directly causes an error: "5" + 3 — because "5" is a string, not an integer

You can check the data type of any variable using Python's built-in type() function:

example.py 
age = 25
print(type(age))
Output
<class 'int'>
04

Type Conversion (Type Casting)

Sometimes you need to convert one data type into another. Python provides built-in functions for this.

type_conversion.py 
# Converting to Integer
num = int("10")
print(num)          # 10

# Converting to Float
value = float(5)
print(value)        # 5.0

# Converting to String
age = 25
text = str(age)
print(text)         # "25"

# Converting to Boolean
print(bool(1))    # True
print(bool(0))    # False
Caution with int() conversion The string you are converting must actually be convertable to an integer. int("Hello") will raise an error because "Hello" is not a number.
05

Algorithms

An algorithm is a step-by-step procedure to solve a problem. Before writing a program, it is always a good idea to think: "What steps should the computer follow to solve this problem?" That sequence of steps is the algorithm.

Example algorithm — Find the sum of two numbers:

1 Start
2 Input the first number
3 Input the second number
4 Add the two numbers
5 Display the result
6 End

Only after defining these steps do we write the actual Python code. Algorithms help you think clearly before coding, break down complex problems, and avoid confusion and errors.

Good programmers plan first Good programmers don't jump straight into coding — they first plan the solution with a clear algorithm.
06

How Variables, Data Types, and Algorithms Work Together

These three concepts connect in every program:

  • Variables — "What information do I have?"
  • Data types — "What kind of information is it?"
  • Algorithms — "What should I do with it?"
Also worth knowing The data types discussed here (int, float, str, bool) store only a single value each. Python also has data types that store collections of data — like lists, sets, tuples, and dictionaries. These are covered in Lesson 13.

Lesson Summary

  • A variable is a named container that stores a value in a program
  • Data types include int (whole numbers), float (decimals), str (text), and bool (True/False)
  • Use type() to check what data type a variable holds
  • Type conversion functions (int(), float(), str()) change data from one type to another
  • An algorithm is a step-by-step plan for solving a problem — always plan before coding
