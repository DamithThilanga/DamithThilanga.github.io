Lesson 9

Mathematical Operators in Python

Mathematical operators allow you to perform calculations in Python — addition, subtraction, multiplication, division, and more. Combined with variables and user input, you can now build programs that solve real mathematical problems.

By the end of this lesson you will be able to

  • Perform basic calculations in Python with integers and floats
  • Understand the different mathematical operators supported in Python
  • Use operators with variables and user input
  • Build a simple calculator program using basic arithmetic operators
01

Basic Mathematical Operators

Python uses similar symbols to normal math, with a few differences:

Operator Meaning Example Result
+Addition5 + 38
-Subtraction5 - 32
*Multiplication5 * 315
/Division5 / 22.5
//Floor Division5 // 22
%Modulus (remainder)5 % 21
**Power (exponent)2 ** 38
operators.py 
print(5 + 3)    # 8
print(10 - 4)   # 6
print(6 * 2)    # 12
print(8 / 2)    # 4.0
Division always returns a float Notice that 8 / 2 gives 4.0, not 4. The / operator always returns a float in Python 3.
02

Special Operators Explained

Floor Division (//) — Removes the decimal part. The actual answer of 5 ÷ 2 is 2.5, but // returns only the whole number part.

print(5 // 2)
Output
2

Modulus (%) — Returns the remainder of a division. When 9 is divided by 2, the whole number result is 4 and the remainder is 1.

print(9 % 2)
Output
1

Power (**) — Raises a number to a power. 2 ** 3 means 2 × 2 × 2.

print(2 ** 3)
Output
8
03

Using Operators with Variables

You can store numbers in variables and calculate with them. You can also store the result in a new variable.

variables.py 
a = 10
b = 5
print(a + b)   # 15
print(a - b)   # 5
print(a * b)   # 50

number1 = 12
number2 = 16
total = number1 + number2
print(total)    # 28
04

Using Operators with User Input

calculator.py 
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
total = num1 + num2
print("Total:", total)
Example run
Enter first number: 5
Enter second number: 3
Total: 8
Forgetting to convert input — string concatenation instead of addition
Wrong 
num1 = input("Enter number: ")
num2 = input("Enter number: ")
print(num1 + num2)   # "23" not 5 !
Correct 
num1 = int(input("Enter number: "))
num2 = int(input("Enter number: "))
print(num1 + num2)   # 5
05

Order of Operations

Python follows standard mathematical rules — multiplication and division happen before addition and subtraction.

print(2 + 3 * 4)   # 14, not 20

Use parentheses to control the order:

print((2 + 3) * 4)   # 20

You can also assign the result of a calculation back to the same variable:

total = 20
total = total + 5
print(total)   # 25

# Or with type conversion:
age = int(age)   # converts age string to integer in-place

Lesson Summary

  • Python has seven arithmetic operators: +, -, *, /, //, %, **
  • The / operator always returns a float; // returns the integer part
  • The % operator returns the remainder of division
  • Always convert input() to int() or float() before doing arithmetic
  • Python follows standard mathematical order of operations — use parentheses to override it
