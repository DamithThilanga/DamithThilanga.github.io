Python Programming Lesson 8 of 20
Lesson 8

From "Hello World" to User Input

Your first programs just displayed fixed text. Now you will write programs that interact with the user — asking for data and using it to produce personalized output.

By the end of this lesson you will be able to

  • Use the input() function to collect data from users
  • Store user input in variables appropriately
  • Convert user input into numerical data types when necessary
  • Build simple interactive Python programs
01

The print() Function Revisited

In the previous lesson, you wrote your first Python program:

print("Hello, World!")

print() is a built-in Python function that displays text on the screen. "Hello, World!" is called a string (text data).

You can print anything:

print("Welcome to Python")
print("I am learning programming!")
print(10)
print(3.14)
02

Using Variables with print()

Instead of typing values again and again, store them in variables and print the variable:

variables.py 
name = "John"
age = 25
print(name)
print(age)
Output
John
25

Imagine you have to print a longer message that appears in several places in the program. Instead of typing the whole text inside print() every time, assign it to a variable and print that variable wherever needed. If you later decide to change the text, you only change it once.

Remember: = means assignment, not equal When assigning a value to a variable, we use =. But it doesn't mean "equal" — it means "the value on the right is assigned to the variable on the left." The double == is used to check equality, which we will cover in Lesson 11.
03

Mixing Text and Variables in print()

You can combine text and variables in one print() call by separating them with commas:

name = "John"
print("Hello", name)
Output
Hello John

Python automatically adds a space between items separated by commas.

04

Taking Input from the User

You can ask the user to enter data using the input() function.

Who is the user? The user is the human who interacts with your program. At the development stage, you are both the programmer and the user. The user sees input prompts and output — but not the code.
greeting.py 
name = input("Enter your name: ")
print("Hello", name)
Example run
Enter your name: John
Hello John
1 input() displays the prompt text and waits for the user to type something
2 Whatever the user types is stored in the variable
3 The program continues with that value
05

Important: input() Always Returns a String

input() always gives you a string (text), even if you type a number.

age = input("Enter your age: ")
print(age)
# If you enter 25, Python still treats it as "25" (a string)
This is a very common beginner mistake If you try to do math with a number entered by the user but forget to convert it, you will get unexpected results or errors. Always convert when you need to use the input as a number.
06

Converting Input to Numbers

If you want to use the input as a number, you must convert it using int() or float():

age.py 
age = int(input("Enter your age: "))
print("You are", age, "years old")

Now age is stored as an integer and you can do arithmetic with it.

07

A Complete Interactive Program

Let's combine everything you've learned:

interactive.py 
name = input("Enter your name: ")
age = int(input("Enter your age: "))
print("Hello", name)
print("You are", age, "years old")
Example run
Enter your name: John
Enter your age: 25
Hello John
You are 25 years old
08

Common Beginner Mistakes

Forgetting quotes around strings
Wrong 
print(Hello)   # NameError
Correct 
print("Hello")
Not converting input when doing arithmetic
Wrong 
age = input("Enter age: ")
print(age + 5)   # TypeError
Correct 
age = int(input("Enter age: "))
print(age + 5)

Lesson Summary

  • print() displays output — separate multiple items with commas
  • Variables store values so you don't have to retype them
  • input() pauses the program and waits for the user to type something
  • input() always returns a string — use int() or float() to convert it to a number
  • You can combine multiple conversions on one line: age = int(input("Enter age: "))
