Lesson 3

Programming Languages

We know that computers only understand 0s and 1s. Programming languages act as a middle layer — letting us write instructions in a human-readable form that are then translated into machine language.

By the end of this lesson you will be able to

  • Define what a programming language is, with clarity on its components
  • Explain why programming languages are needed in relation to practical requirements
  • Distinguish between different categories of programming languages according to different criteria
  • Identify examples of popular programming languages and their common uses
01

What Is a Programming Language?

A programming language is a formal language used by humans to write instructions that a computer can understand and execute.

As humans, we communicate using natural languages such as English or Sinhala. However, computers cannot understand these languages directly. Computers only understand machine language, which consists of 0s and 1s. Writing instructions directly in machine language is extremely difficult, time-consuming, and error-prone.

Programming languages act as a middle layer between humans and computers. They allow us to write instructions in a human-readable form, which are then translated into machine language so the computer can execute them. Your beautiful photo on a mobile phone, that action movie on a laptop screen, that online transaction — all these are actually 0s and 1s in electronic circuits. What makes these boring 0s and 1s into interesting real-life events are programming languages.

Examples of popular programming languages include:

  • Python
  • Java
  • C
  • JavaScript
  • C++

Each programming language has its own rules (called syntax) and its own way of writing instructions.

02

Why Do We Need Programming Languages?

Programming languages are needed because they make programming:

  • Easier
  • More readable
  • Less error-prone
  • More powerful

Without programming languages, creating even a simple program would require writing thousands of 0s and 1s, which is not practical for humans. Programming languages allow us to focus on solving problems, rather than worrying about how the computer hardware works internally.

03

Classification Based on Level of Abstraction

This classification is based on how close a language is to human thinking or computer hardware.

Low-Level Languages — These languages are very close to the hardware.

  • Difficult for humans to read and write
  • Very fast and efficient
  • Examples: Machine language, Assembly language, C (to some extent)

High-Level Languages — These languages are closer to human language and thinking.

  • Easy to learn and use
  • Less concerned with hardware details
  • Examples: Python, Java, JavaScript
Python is a high-level language This is one reason Python is ideal for beginners — it lets you focus on solving problems without worrying about memory management or hardware details.
04

Classification Based on Programming Paradigm

A programming paradigm is a style or approach to writing programs.

Procedural Programming Languages — Programs are written as a series of step-by-step instructions. Focus on procedures or functions. Examples: C, Python (supports this style).

Object-Oriented Programming Languages — Programs are organized around objects that represent real-world entities. Focus on data and behaviour together. Examples: Java, C++, Python.

Functional Programming Languages — Programs are written using mathematical functions. Focus on what to do, not how to do it. Examples: Haskell, Python (supports some functional concepts).

Python is multi-paradigm Python supports procedural, object-oriented, and functional programming styles, making it very flexible.
05

Classification Based on Execution

This classification depends on how code is converted into machine language.

Compiled Languages — The entire program is converted into machine code before execution.

  • Faster execution
  • Errors detected before running
  • Examples: C, C++

Interpreted Languages — Code is executed line by line.

  • Easier to test and debug
  • Slightly slower execution
  • Examples: Python, JavaScript
06

Classification Based on Purpose

General-Purpose Languages — Can be used for many types of applications.

  • Python: desktop apps, web apps, machine learning and AI, web scraping, game development
  • Java: desktop apps, Android apps, web application backend
  • C++: game development, desktop applications

Domain-Specific Languages — Designed for a specific field or purpose.

  • SQL — databases
  • R — statistics
  • HTML — web page structure
07

Practical Uses of Programming Languages

Programming languages are used in almost every area of modern life:

  • System software — Operating systems like Windows and Android are written with low-level languages that can directly interact with hardware
  • Application software — Mobile apps like WhatsApp, desktop apps like Microsoft Word and Google Chrome
  • Web development — Front-end with HTML and JavaScript, back-end with Python, Java, or PHP
  • Data analysis, machine learning, and AI — Predicting disease trends, recommendation systems (Netflix, YouTube)
  • Automation and scripting — Organizing files, generating reports, sending automated emails
  • Game development — C++ is popular for high-graphics games due to its speed and hardware access

Lesson Summary

  • A programming language is a tool that allows humans to communicate instructions to computers
  • Languages are classified by level of abstraction, paradigm, execution method, and purpose
  • Python is a high-level, interpreted, general-purpose, multi-paradigm language
  • Different programming languages exist for different requirements — choose wisely
  • Python is ideal for beginners because of its simple, readable, English-like syntax
