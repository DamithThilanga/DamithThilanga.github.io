Project: Functions and Error Handling

In the previous activities, you first built programs using functions, parameters, return values, and modules — then enhanced those same programs using try and except. This is how real programs are often developed: make it work, then improve its reliability. In this lesson, we discuss both stages for each project.

In this lesson you will see

  • How each project is built with functions first
  • How error handling is layered on top to improve reliability
  • The programming mindset: build → improve → refine

Step 2 — With error handling 
def greet_user(name):
    print(f"Hello {name}")

try:
    name = input("Enter your name: ")
    if name == "":
        print("Name cannot be empty")
    else:
        greet_user(name)
except:
    print("Something went wrong")
Improvement — checks for empty input.

Project 15 — ATM Simulator

A simple ATM with a menu loop, using a global variable for balance.

Step 1 — Function solution 
balance = 10000

def check_balance():
    print("Balance =", balance)

def deposit(amount):
    global balance
    balance += amount
    print("Deposit successful")

def withdraw(amount):
    global balance
    balance -= amount
    print("Withdrawal successful")

while True:
    print("1. Check Balance")
    print("2. Deposit")
    print("3. Withdraw")
    print("4. Exit")
    choice = input("Enter choice: ")

    if choice == "1":    check_balance()
    elif choice == "2":
        amount = float(input("Enter deposit amount: "))
        deposit(amount)
    elif choice == "3":
        amount = float(input("Enter withdraw amount: "))
        withdraw(amount)
    elif choice == "4":
        print("Thank you")
        break
    else:
        print("Invalid choice")
Problem in basic version The user can withdraw more than the available balance, and entering invalid numbers will crash the program. Both issues are fixed in Step 2.
Step 2 — Enhanced (handles overdraw and invalid input) 
balance = 10000

def check_balance():
    print("Balance =", balance)

def deposit(amount):
    global balance
    balance += amount
    print("Deposit successful")

def withdraw(amount):
    global balance
    if amount > balance:
        print("Insufficient balance")
    else:
        balance -= amount
        print("Withdrawal successful")

while True:
    try:
        print("1. Check Balance")
        print("2. Deposit")
        print("3. Withdraw")
        print("4. Exit")
        choice = input("Enter choice: ")

        if choice == "1":    check_balance()
        elif choice == "2":
            amount = float(input("Enter deposit amount: "))
            deposit(amount)
        elif choice == "3":
            amount = float(input("Enter withdraw amount: "))
            withdraw(amount)
        elif choice == "4":
            print("Thank you")
            break
        else:
            print("Invalid choice")
    except ValueError:
        print("Please enter valid amount")
Improvements — now handles invalid amounts, overdrawing, and keeps the menu loop running safely.
