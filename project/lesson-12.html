Exercise 1 — Check if a Number Is Positive

Question: Write a program that checks if a number is positive.

exercise1.py 
number = float(input("Enter a number: "))
if number > 0:
    print("Positive number")

The program gets a number from the user and checks if it is greater than 0. If true, it prints "Positive number".

Sample run

Positive number

Exercise 2 — Even or Odd

Question: Check whether a number is even or odd.

exercise2.py 
number = int(input("Enter a number: "))
if number % 2 == 0:
    print("Even number")
else:
    print("Odd number")

The % operator gives the remainder. If number % 2 equals 0, the number is divisible by 2 and therefore even.

Sample run
Enter a number: 9
Odd number

Exercise 3 — Age Categorizer

Question: Categorize age as Child (<13), Teen (13–19), Adult (20+), or Invalid (>120).

exercise3.py 
age = int(input("Enter age: "))
if age > 120:
    print("Invalid age")
elif age < 13:
    print("Child")
elif age <= 19:
    print("Teen")
else:
    print("Adult")
Important: check invalid age first If age is 150, it is also greater than 20. Without checking the invalid condition first, the program might incorrectly print "Adult" for an age of 150.
Sample run
Enter age: 15
Teen

Exercise 4 — Divisibility by 5

Question: Check if a number is divisible by 5.

exercise4.py 
number = int(input("Enter number: "))
if number % 5 == 0:
    print("Divisible by 5")
else:
    print("Not divisible by 5")

If the remainder after dividing by 5 is zero, the number is divisible by 5.

Sample run
Enter number: 25
Divisible by 5

Exercise 5 — Compare Two Numbers

Question: Compare two numbers and print which is larger, or if they are equal.

exercise5.py 
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))
if num1 > num2:
    print("First number is larger than the second number")
elif num2 > num1:
    print("Second number is larger than the first number")
else:
    print("Two numbers are equal")

There are exactly three possibilities: first larger, second larger, or equal. Three branches handle all three cases.

Sample run
Enter first number: 10
Enter second number: 10
Two numbers are equal

Exercise 6 — Username and Password Check

Question: Check username and password. Correct credentials are username: JohnW, password: JohN#W123.

exercise6.py 
username = input("Enter username: ")
password = input("Enter password: ")
if username == "JohnW" and password == "JohN#W123":
    print("Welcome John")
else:
    print("Username or password incorrect. Access denied")

Uses the logical operator and — both conditions (correct username AND correct password) must be true simultaneously.

Sample run
Enter username: JohnW
Enter password: JohN#W123
Welcome John

Exercise 7 — Divisibility by 2 and 3

Question: Check divisibility by 2 and 3 with four possible outputs.

exercise7.py 
number = int(input("Enter number: "))
if number % 2 == 0 and number % 3 == 0:
    print("Divisible by both 2 and 3")
elif number % 2 == 0:
    print("Divisible only by 2")
elif number % 3 == 0:
    print("Divisible only by 3")
else:
    print("Not divisible by either")
Check the most specific condition first If a number is divisible by both 2 and 3, it is also individually divisible by each. By checking "both" first, we avoid incorrectly printing "only by 2" for a number like 12 which is also divisible by 3.
Sample run
Enter number: 12
Divisible by both 2 and 3

Exercise 8 — Bill Splitting with Discount

Question: Split a restaurant bill equally among friends. Apply a 10% discount if the bill is over 5000, or 5% if over 2000.

exercise8.py 
bill = float(input("Enter total bill amount: "))
people = int(input("Enter number of people: "))

if bill > 5000:
    discount = bill * 0.10
elif bill > 2000:
    discount = bill * 0.05
else:
    discount = 0

final_bill = bill - discount
per_person = final_bill / people
print("Discount:", discount)
print("Final bill:", final_bill)
print("Each person pays:", per_person)
Sample run
Enter total bill amount: 6000
Enter number of people: 4
Discount: 600.0
Final bill: 5400.0
Each person pays: 1350.0

Lesson Summary

  • Always check the most specific or most extreme condition first in an if-elif chain
  • Use and when multiple conditions must all be true simultaneously
  • The % operator is very useful for checking divisibility
  • In Python, conditions are evaluated in order — once a True condition is found, the rest are skipped
  • Your code doesn't need to look exactly like sample solutions — many different solutions can solve the same problem
