Lesson 7

Setting Up Python: Installation and Development Environment

Before writing real Python programs, you need to set up Python on your computer. This lesson walks you through installing Python, running your first program, and setting up Visual Studio Code as your IDE.

By the end of this lesson you will be able to

  • Install Python on your computer (Windows or macOS)
  • Verify that Python is installed correctly
  • Set up Visual Studio Code for Python programming
  • Run a basic Python script successfully
01

Installing Python on Windows

As of 2026, the latest stable Python version is from the Python 3.x series (around 3.13+).

1
Download Python — Go to the official website: https://www.python.org and click Download Python (latest version).
2
Run the Installer — Open the downloaded .exe file. IMPORTANT: Tick the checkbox "Add Python to PATH" before clicking Install Now.
3
Verify Installation — Open Command Prompt and type:
python --version
Output
Python 3.13.x
Don't skip "Add Python to PATH" If you forget to tick "Add Python to PATH", your computer won't be able to find Python when you type commands in the terminal.
02

Installing Python on macOS

Most Macs already have Python installed, but usually an older version. Install the latest one.

1
Download Python — Go to https://www.python.org and download the macOS installer (.pkg file).
2
Install Python — Open the downloaded .pkg file and follow the installation steps.
3
Verify Installation — Open Terminal and type python3 --version. You should see the installed version.
03

Running Your First Program (Without an IDE)

Before using advanced tools, it's important to understand how Python works with a simple text editor.

1 Open Notepad (Windows) or TextEdit (Mac)
2
Write the following code: 
print("Hello, World!")
3
Save the file as hello.py. Make sure the file type is set to "All Files" and the extension is .py.
4
On Windows: Open Command Prompt, navigate to the file location with cd Desktop, then run python hello.py.
On macOS: Open Terminal, navigate to the file location, then run python3 hello.py.
Output
Hello, World!

Congratulations! You just ran your first Python program.

04

Why Do We Need an IDE?

Writing code in Notepad works, but it is not practical for real development. An IDE (Integrated Development Environment) provides:

  • Syntax highlighting — different colors for different code elements
  • Error detection — underlines mistakes before you run the code
  • Auto-completion — suggests code as you type
  • Easy execution — run your code with one click or keyboard shortcut

You don't get any of these facilities with Notepad or ordinary text editors.

05

Installing Visual Studio Code (VS Code)

VS Code is one of the most popular and beginner-friendly IDEs.

1
Download VS Code — Go to https://code.visualstudio.com and download the version for your operating system.
2
Install VS Code — Run the installer and follow the steps.
3
Install the Python Extension — Open VS Code, go to the Extensions panel (left sidebar), search for "Python", and install the extension by Microsoft.
4
Open a folder — Create a folder for your Python projects, then open it in VS Code with File > Open Folder.
5
Create and run a file — Create a new file called hello.py, write print("Hello, World!"), and press the Run button (or press F5).
Recommended workflow Always keep one folder for your Python course files. Name your files clearly (e.g. lesson08_input.py) so you can find them later.

Lesson Summary

  • Python is downloaded from python.org — always tick "Add Python to PATH" on Windows
  • Verify installation by typing python --version in the terminal
  • Python files have the .py extension and are run from the terminal or an IDE
  • VS Code is a free, beginner-friendly IDE that adds syntax highlighting, error detection, and auto-completion
